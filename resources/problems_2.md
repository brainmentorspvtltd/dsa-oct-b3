# Easy to Moderate DSA Practice Problems  
## For Java, Python, JavaScript, Dart Students  
  
These 30 problems are perfect for students starting their DSA journey. They cover number logic, digit operations, conditions, and simple loops — all meant to sharpen logic building.  
  
---  
  
### Number and Digit-Based Logic  
  
1. **Print All Digits of a Number**  
   Extract and print each digit without using string functions.  
  
2. **Sum of All Digits**  
   Input: `3471` → Output: `15`  
  
3. **Reverse a Number**  
   Input: `1234` → Output: `4321`  
  
4. **Count Number of Digits**  
   Input: `5402` → Output: `4`  
  
5. **Check if Number is Palindrome**  
   Input: `121` → True, Input: `123` → False  
  
6. **Check if Number is Prime**  
   Input: `13` → True, `12` → False  
  
7. **Check if Number is Even or Odd**  
  
8. **Find the Largest Digit in a Number**  
  
9. **Swap First and Last Digit**  
   Input: `1234` → Output: `4231`  
  
10. **Check if All Digits Are Unique**  
  
---  
  
### Loops and Series  
  
11. **Print Numbers from 1 to N**  
  
12. **Print All Even Numbers from 1 to N**  
  
13. **Print Sum of First N Natural Numbers**  
  
14. **Print Multiplication Table of a Number**  
    Example: `5 → 5 10 15 ... 50`  
  
15. **Print Factorial of a Number**  
    Input: `5` → Output: `120`  
  
16. **Fibonacci Series up to N Terms**  
  
17. **Count Even and Odd Digits Separately**  
  
18. **Sum of Squares of First N Numbers**  
  
19. **Print All Prime Numbers from 1 to N**  
  
20. **Count How Many Digits are Divisors of the Number**  
    Input: `122` → Output: `3` (1,2,2 divide 122)  
  
---  
  
### Logical Analysis and Decision Making  
  
21. **Check if Digits are in Ascending Order**  
    Input: `1239` → True, `321` → False  
  
22. **Check if Number is Armstrong**  
    Input: `153 = 1³ + 5³ + 3³` → True  
  
23. **Check if Number is a Harshad Number**  
    (Divisible by sum of digits)  
  
24. **Frequency of Each Digit in a Number**  
    Input: `2447` → Output: `2:1, 4:2, 7:1`  
  
25. **Calculate Digital Root of a Number**  
    Keep summing digits until a single digit remains  
  
---  
  
### Transformations and Rotations  
  
26. **Rotate Number Right by K Digits**  
    Input: `12345`, k = 2 → Output: `45123`  
  
27. **Rotate Number Left by K Digits**  
    Input: `12345`, k = 2 → Output: `34512`  
  
28. **Flip Digits from Both Ends to New Number**  
    Input: `32145` + Reverse → `54321` → Flip = `12543`  
  
29. **Form Max and Min Numbers Using Digits**  
    Input: `4312` → Output: `4321` and `1234`  
  
30. **Check if Number is a Special Number**  
    Sum of factorial of digits divides original number  
  
---  
  
**Note**: All problems should be implemented using loops and conditionals. Avoid using built-in shortcuts like string manipulation or libraries unless solving with alternate logic.
